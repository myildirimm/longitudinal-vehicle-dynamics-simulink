# Summary

Longitudinal Vehicle Dynamics is an open MATLAB/Simulink model for simulating vehicle motion in the direction of travel. The model represents the main components of longitudinal vehicle behaviour, including traction, braking, aerodynamic drag, rolling resistance, wheel rotational dynamics, tire slip, and speed response. It is intended for engineering education, vehicle dynamics analysis, and early-stage control design studies.

# Statement of need

Longitudinal vehicle dynamics models are widely used in automotive engineering education, braking and traction studies, vehicle control design, and autonomous driving research. Students and researchers often require a transparent, modifiable, and reusable simulation model that can be inspected and adapted for different vehicle parameters and driving scenarios. This software provides an accessible MATLAB/Simulink implementation that can be used as a starting point for longitudinal vehicle dynamics simulations and speed-control studies.

The model was originally shared through MATLAB Central File Exchange and has received substantial user engagement from the MATLAB/Simulink community. This repository formalizes the model as an open and citable research software contribution with documentation and reproducible example structure.

# Software description

The software provides a modular Simulink implementation of a longitudinal vehicle dynamics model. The model includes a longitudinal force balance, aerodynamic drag, rolling resistance, wheel rotational dynamics, longitudinal tire slip, and a simplified Pacejka-type tire-force representation. The repository also includes documentation of the main governing equations and is structured to support example scripts, validation cases, and future extensions.

# Applications

The model can be used for:

* vehicle acceleration simulation,
* braking-distance analysis,
* traction-force and braking-force studies,
* longitudinal tire slip analysis,
* introductory vehicle dynamics teaching,
* early-stage speed-control and autonomous driving studies.

# Availability

The software is available as an open MATLAB/Simulink model through this GitHub repository. The original model was also shared through MATLAB Central File Exchange.
