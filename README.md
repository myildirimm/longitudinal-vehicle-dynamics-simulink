# Longitudinal Vehicle Dynamics Simulink Model

An open MATLAB/Simulink model for longitudinal vehicle dynamics, traction, braking, and speed-control studies.

## Overview

This repository provides a MATLAB/Simulink implementation of a longitudinal vehicle dynamics model. The model can be used to simulate vehicle acceleration, braking, resistive forces, wheel dynamics, and speed response under different driving conditions.

The project is intended for engineering education, vehicle dynamics analysis, and early-stage control design studies.

The original model was first shared through MATLAB Central File Exchange and has been used by the MATLAB/Simulink community for vehicle dynamics simulations.

## Features

- Longitudinal vehicle dynamics simulation
- Traction and braking scenarios
- Aerodynamic drag and rolling resistance
- Road slope / grade resistance
- Wheel speed and vehicle speed response
- Reusable MATLAB/Simulink structure for education and research
